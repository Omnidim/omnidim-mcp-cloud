# Security policy

## Reporting a vulnerability

Email `security@omnidim.io` with a description, reproduction steps, and the affected version. We aim to acknowledge within two business days.

Please do not file public issues for security reports.

## Scope

In scope: the OAuth provider endpoints, the MCP transport, token handling, and any code in this repository.

Out of scope: vulnerabilities in upstream dependencies (please report those to their maintainers and include the CVE in your message), denial of service attacks against the public service, or issues in unrelated OmniDim products.
